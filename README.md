# Wahlblock-UI

Singlepage Application built with Typescript, React and Bulma CSS.

Prerequisites:  
 - NodeJS
 - Yarn  
  
To use this project to this point clone the repository and run  
`yarn`  
`yarn start`  
To build for production enviroment use  
`yarn build`

## Frameworks used:
 - Apollo
 - Bloomer
 - Chart.js
 - Jest
 - React
 - Redux
 - React-Router
 - Typescript
 - Webpack
